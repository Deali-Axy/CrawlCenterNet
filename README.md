# CrawlCenter

可视化的分布式爬虫管理工具（开发中）

## 技术

- Asp.Net Core
- FreeSql
- MongoDB
- Exceptionless
- Hangfire
- [Hangfire.Redis.StackExchange](https://github.com/marcoCasamento/Hangfire.Redis.StackExchange)
- OpenApi
- X.PagedList
